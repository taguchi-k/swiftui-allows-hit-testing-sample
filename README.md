# swiftui-allows-hit-testing-sample

`allowsHitTesting()` を利用して、 `Gesture` が発火するかどうかをハンドリングするサンプル

* `Toggle` がONのときは `Text` のViewをタップすると背景色が切り替わる
* `Toggle` がOFFの場合は `Text` のViewをタップしても何も起きない
